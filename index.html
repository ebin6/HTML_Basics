What is Python

Python is a high-level, general-purpose programming language that emphasizes code readability and supports multiple paradigms. It was created by Guido van Rossum.

Why learn Python ?

  • It is easy to learn
  • General purpose programming language
  • It is a dynamicaly typed programming language
